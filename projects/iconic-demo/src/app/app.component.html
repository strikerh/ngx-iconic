ngx-iconic

A standalone Angular icon component for Material Design icons

Installation

npm install ngx-iconic

Setup


import { NgxIconicModule } from 'ngx-iconic';

      @NgModule({
        imports: [
        NgxIconicModule.forRoot({
      // Optional configuration
      defaultIcon: 'all_out',
      iconBasePath: 'assets/icons',
      flipInRtl: true
    })
  ]
})
      export class AppModule { }

Basic Usage

mat_twoTone home mat_twoTone face face 

<iconic icon="face"></iconic>
<iconic  icon="face" class="text-xl"></iconic>
<iconic icon="favorite"  color="red"  class="text-xl"></iconic>
<iconic icon="star" color="#FFD700"  class="text-xl"></iconic>
<iconic  class="text-xl">mat_twoTone home </iconic>
<iconic  class="text-xl" icon="home">mat_twoTone face face</iconic> // Priority to the icon property

Note on Content Projection: You can use content projection to define both icon style and name. The format is <iconic>STYLE_ALIAS ICON_NAME</iconic> where STYLE_ALIAS is one of the supported style aliases and ICON_NAME is the Material icon name.

Priority Rules: When both content projection and properties are used, the component properties take priority. For example, if you specify icon="home" in the property and mat_twoTone face in content, "home" will be used as the icon name.

Style Aliases

You can use these aliases in content projection or as values for the iconStyle property:

Icon Style Aliases
materialIcons mi, mif, miFill, mi_fill, mat_fill
materialIconsOutlined mio, miOutlined, mi_outlined, mat_outlined, mat_outline
materialIconsRound mir, miRound, mi_round, mat_round
materialIconsSharp mis, miSharp, mi_sharp, mat_sharp
materialIconsTwoTone mit, miTwoTone, mi_twoTone, mat_twoTone

RTL Support

Icons automatically flip in RTL context when flipInRtl is enabled in the configuration.

Available Size Classes

You can set the icon size using either the size property or by adding one of these CSS classes:

  • text-xs - Extra small (0.75rem)
  • text-sm - Small (0.875rem)
  • text-base - Base (1rem)
  • text-lg - Large (1.125rem)
  • text-xl - Extra large (1.25rem)
  • text-2xl - 2X large (1.5rem)
  • text-3xl - 3X large (1.875rem)
  • text-4xl - 4X large (2.25rem)
  • text-5xl - 5X large (3rem)
  • text-6xl - 6X large (3.75rem)
  • text-7xl - 7X large (4.5rem)
  • text-8xl - 8X large (6rem)
  • text-9xl - 9X large (8rem)

API Reference

Inputs

Name Type Default Description
icon IconNames 'all_out' Name of the Material Design icon
iconStyle IconStyle 'materialIcons' Icon variant style
flip boolean undefined Manually flip the icon horizontally
color string 'currentColor' Icon color (CSS color value)
size IconSizes undefined Icon size from xs to 9xl

Configuration

Name Type Default Description
defaultIcon string 'all_out' Default icon when none specified
iconBasePath string 'assets/icons' Base path for icon assets
flipInRtl boolean true Auto-flip directional icons in RTL
